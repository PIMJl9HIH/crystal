Key Features

Our goal is to make total decontamination of the air/water syringe a common practice by making it easy, maybe even a little fun. Crystalgenics® Evolve was designed to conform to the new Integrated Cross Infection Control (ICIC) specification.

  • Factory-Installed Tip Retainer
    Crystalgenics® Evolve is the only air/water syringe system that comes already fitted with a disposable air/water syringe tip retainer. This eliminates the need to buy or source an adapter to conform to infection control protocols.
  • One Touch SafetyTM
    Crystalgenics® Evolve uses a One Touch Safety™ system to easily remove the Evolve sleeve between patients. The air/water syringe is the only item in a dental office that is used on very patient but never sterilized. The One Touch Safety™ system solves this issue and ensures each patient receives a safe treatment.
